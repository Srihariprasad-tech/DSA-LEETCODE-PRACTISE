# 📊 Array & String - Pattern Based DSA

This folder contains problems related to **Arrays and Strings**, organized based on important **DSA problem-solving patterns**.

---

## 📂 Patterns Covered

* Two Pointers
* Sliding Window
* Prefix Sum
* Kadane’s Algorithm
* Hashing (Frequency Map)

---

## 🧠 Approach

Instead of solving problems topic-wise, this folder follows a **pattern-based approach**, which helps in:

* Identifying patterns quickly
* Applying similar logic to multiple problems
* Improving problem-solving skills for interviews

---

## 📌 Organization

Each subfolder represents a **specific pattern**, and contains:

* Array-based problems
* String-based problems
